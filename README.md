# Bash Color Config

When SSHing into different servers, the shell color configuration may vary across hosts.  
Reconfiguring the terminal colors every time can be tedious.

This repository provides a simple configuration file to keep the shell text and directory colors consistent across different machines.

## Features

- Unified shell text color
- Consistent directory color in `ls`
- Easy to apply on any Linux host
